🚀 SAR Narrative Generator

AI-powered system for automated Suspicious Activity Report (SAR) narrative generation using structured financial case data.

📌 Problem Statement

Financial institutions must generate Suspicious Activity Reports (SARs) manually, which is:

Time-consuming

Error-prone

Compliance-sensitive

Our system automates SAR narrative generation using AI while maintaining auditability and security.
